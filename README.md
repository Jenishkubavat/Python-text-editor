# Python-text-editor

Python Text Editor using Tkinter with tab functionality.

Made using Python Python 3.10.4 on Windows

![picture alt](https://github.com/Jenishkubavat/Python-text-editor/blob/master/Screenshot%202022-05-12%20145658.png?raw=true)

## Menus & Functionality:

* ### File:  
  * New  
  * Open  
  * Save As...  
  * Close  
  * Exit  

* ### Edit and Right click in text window:  
    * Undo  
    * Cut  
    * Copy  
    * Paste  
    * Delete  
    * Select All  
    
* ### Format:  
    * Word Wrap  
    
* ### Tabs:  
    * Right Click  
      * New Tab Option  
    * Center Click  
      * Close Tab  
    * Drag  
      * Rearrange Tab  
      
* ### Keyboard Shortcuts:  
    * Ctrl-s: Save  
    * Ctrl-o: Open  
    * Ctrl-w: Close tab  
    * Ctrl-n: New tab  
    * Ctrl-tab: Next tab  
    * Ctrl-shift-tab: Previous tab  
    * Ctrl-u: Undo  
    * Ctrl-y: Redo
